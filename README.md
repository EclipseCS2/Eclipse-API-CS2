# Eclipse-API-CS2
This API provides access to game Counter-Strike 2 state data by reading specific memory addresses (offsets). It allows for the retrieval of player information, world state, and entity data.

Данный API предоставляет доступ к данным о состоянии игры Counter-Strike 2 путем чтения определенных адресов памяти (оффсетов). Он позволяет получать информацию об игроках, состоянии мира и данных сущностей.
